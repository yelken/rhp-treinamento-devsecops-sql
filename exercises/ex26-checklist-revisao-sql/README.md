# Ex26 — Checklist de revisão de consultas (segurança e performance)

## Objetivo
Adotar uma checklist mínima para revisão de consultas SQL focada em segurança, privacidade e performance.

## Passos
1. Adapte `CHECKLIST_REVIEW_SQL.template.md` às necessidades do time.
2. Inclua a checklist no template de PR ou como comentário padrão nas revisões.
3. Aplique a checklist em 1 PR real (ou simulado) neste repositório.

## Critério de aceite
- Checklist incorporada ao fluxo de revisão e usada em pelo menos 1 mudança.

## Nota: PII e PHI
- A checklist reforça práticas consistentes de proteção de PII/PHI e metas de performance.
