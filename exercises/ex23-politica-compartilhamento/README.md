# Ex23 — Política de compartilhamento de dados (export responsável)

## Objetivo
Formalizar critérios para export de dados com segurança (PII/PHI), minimização e trilha de auditoria.

## Passos
1. Preencha `DATA_SHARING_POLICY.template.md` com regras de export (o que, para quem, como, por quanto tempo).
2. Defina quando é obrigatório usar views mascaradas e anonimização.
3. Defina logging/auditoria mínima do processo de export.

## Critério de aceite
- Documento preenchido, versionado, revisado (PR) e referenciado nos exercícios de export (Ex20).

## Nota: PII e PHI
- Toda export deve justificar necessidade de PII/PHI e aplicar controles (mascaramento/anonimização) por padrão.
